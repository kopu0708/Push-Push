# 🥊 PUSH-PUSH (2D 물리 밀어내기 서바이벌)

![게임 플레이 화면](스크린샷이나_GIF_플레이_이미지_링크)

**"30초의 제한 시간, 차지 대시로 상대를 원판 밖으로 날려버리세요!"**
유니티 2D 물리 엔진과 코루틴을 활용하여 짜릿한 반동과 타격감을 구현한 탑뷰 아케이드 액션 게임입니다.

---

## 🎮 게임 룰 (Game Rules)
* **제한 시간:** 30초 동안 치열한 밀어내기 공방전이 펼쳐집니다.
* **승리 조건:** 스페이스바로 기를 모아 대시하여 상대를 원판(Arena) 밖으로 더 많이 떨어트린 쪽이 승리합니다.
* **장외 리스폰:** 링 밖으로 떨어지면 상대방이 점수를 획득하며, 떨어진 캐릭터는 즉시 시작 위치로 리스폰되어 전투를 재개합니다.

---

## 🛠️ 기술 스택 (Tech Stack)
* **Engine:** Unity (2D)
* **Language:** C#
* **Version Control:** Git / GitHub

---

## 💡 핵심 구현 기능 (Key Features)

### 1. 예외 처리가 적용된 차지 대시 (Charge Dash) 메커니즘
* 물리적 관성에 의해 캐릭터가 미끄러지거나 움직이는 도중에는 기를 모을 수 없도록 **`rb.linearVelocity.magnitude < 0.1f` 예외 조건**을 설정하여 공정한 턴제 공방전 느낌의 규칙을 확립했습니다.
* 스페이스바(플레이어) 및 코루틴 타이머(적)를 통해 누르고 있는 시간에 비례하여 힘(`currentForce`)이 충전되도록 설계했습니다.

### 2. 직관적인 UI 게이지 연동
* 플레이어뿐만 아니라 적 캐릭터의 머리 위에도 `World Space Canvas`를 배치하여, 실시간 충전 능력을 `UI Image (Fill Amount)`와 연동해 직관적으로 시각화했습니다. 이를 통해 상대방의 공격 타이밍을 예측하는 심리전 요소를 강화했습니다.

### 3. 코루틴(Coroutine) 기반의 상태식 적 AI 구현
* 초급 AI(Beginner) 패턴을 일반 `Update` 문법 대신 **`IEnumerator`와 `WaitUntil` 코루틴**으로 구현하여 불필요한 매 프레임 연산을 최적화했습니다.
* 적 AI 역시 플레이어와 동일하게 완전히 멈춘 상태를 감지한 후 기를 모아 풀 차지 돌격을 감행하도록 밸런스를 맞추었습니다.

### 4. 에러 방어 코드 (MissingReferenceException 예외 처리)
* 캐릭터가 장외로 나갔을 때 오브젝트가 파괴되거나 리스폰되는 과정에서, 적 AI가 이미 파괴된 플레이어의 위치(`target.position`)를 참조하여 발생하는 **`MissingReferenceException`을 방지**하기 위해 돌진 직전 Target의 `null` 상태를 재검증하는 안전장치를 구축했습니다.
