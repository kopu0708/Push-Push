#게임 이름: PUSH-PUSH
## 게임 룰 (Game Rules)
* **제한 시간:** 30초
* **승리 조건:** 원판 위에서 스페이스바로 기를 모아 적을 들이받고, 링 밖으로 더 많이 떨어트린 쪽이 승리합니다.
* **리스폰:** 링 밖으로 떨어지면 점수를 잃고 즉시 중앙으로 리스폰되어 전투를 이어갑니다.

## 기술 스택 (Tech Stack)
* **Engine:** Unity (2D)
* **Language:** C#

## 핵심 구현 기능 ##
1.완전히 멈춘 상태(`velocity < 0.1f`)에서만 기를 모을 수 있도록 예외 처리를 적용했습니다.
2.누르고 있는 시간에 비례하여 힘이 충전되며, `UI Image (Fill Amount)`와 연동하여 플레이어와 적dml 직관적인 게이지 바를 구현했습니다.
3.코루틴을 사용하여 플레이어의 위치를 추적하고 일정 시간 대기 후 돌진하는 적 AI를 `IEnumerator`와 `WaitUntil`을 활용해 구현했습니다.
4.플레이어 파괴 시 발생할 수 있는 `MissingReferenceException` 에러를 방지하기 위해 돌진 직전 Target null 체크 방어 코드를 적용했습니다.

###개발 목표: 
1.싱글톤 구조로 게임매니저를 구현하고 게임이 끝났을 때와 점수를 관리하는 기능 구현합니다.
2.배경음악과 효과음을 입힘니다.
3.중급 / 고급 AI 패턴 추가 합니다.(도망가기, 짤짤이 공격 등)
4.2인 멀티플레이어 기능 구현
